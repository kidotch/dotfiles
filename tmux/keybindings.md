# tmux キーバインド一覧

## コピーモード

| キー | 動作 |
|---|---|
| `Alt+Shift+Z` | コピーモードに入る |
| `v` | 選択開始 |
| `Space` | 選択開始 |
| `y` | 選択範囲をクリップボードにコピー (win32yank) |

## スクロール

| キー | 動作 |
|---|---|
| `Alt+U` | 1行上スクロール |
| `Alt+D` | 1行下スクロール |
| `Ctrl+Alt+U` | 半ページ上スクロール |
| `Ctrl+Alt+D` | 半ページ下スクロール |
| `Alt+PgUp` | 1ページ上スクロール |
| `Alt+PgDn` | 1ページ下スクロール |

## ペイン移動

| キー | 動作 |
|---|---|
| `Alt+Shift+H` | 左のペインへ移動 |
| `Alt+Shift+J` | 下のペインへ移動 |
| `Alt+Shift+K` | 上のペインへ移動 |
| `Alt+Shift+L` | 右のペインへ移動 |
| `Alt+O` | 次のペインへ移動 |
| `Alt+Shift+O` | 前のペインへ移動 |

## ペイン分割

| キー | 動作 |
|---|---|
| `Alt+Shift+%` | 水平分割 (カレントディレクトリで開く) |
| `Alt+Shift+"` | 垂直分割 (カレントディレクトリで開く) |

## その他

| キー | 動作 |
|---|---|
| `prefix + q` | ペイン番号を表示 (2秒間) |
